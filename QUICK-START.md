## 📱 Installation auf Mamas Chromebook

### Schritt 1: Zur Website gehen
1. Öffne **Chrome** auf dem Chromebook
2. Gehe zu: `https://lucsaan.github.io/spielesammlung/spieleseite.html`

### Schritt 2: Als App installieren
1. Klicke auf das **Menü** (⋮) oben rechts
2. Wähle **"App installieren..."** oder **"Spieleseite installieren..."**
3. Klicke auf **"Installieren"**

### Schritt 3: Fertig! 🎉
- Die App erscheint im **App-Launcher** (Kreis-Symbol unten links)
- Mama kann sie wie jede andere App öffnen
- Ein Klick → Spieleauswahl → Mahjong spielen! 🀄

---

## 🔐 Passwörter speichern

Wenn eine Spieleseite Login benötigt:
1. Melde dich beim ersten Besuch an
2. Chrome fragt: **"Passwort speichern?"**
3. Klicke **"Speichern"**
4. Ab jetzt automatisches Login! ✅

**Wichtig**: Die meisten Spiele brauchen KEIN Login!

## 🛠️ Updates & Änderungen

### Neue Spiele hinzufügen

1. **Lokal bearbeiten**: Öffne `spieleseite.html`
2. **Änderungen machen**: Neue Spiele-Links einfügen
3. **Zu GitHub pushen**:
   ```bash
   git add .
   git commit -m "Neue Spiele hinzugefügt"
   git push
   ```
4. **Warten**: 1-2 Minuten, dann ist die Seite aktualisiert!

### Service Worker Cache leeren (nach Updates)

Falls Änderungen nicht sofort sichtbar sind:
1. Öffne Chrome DevTools (F12)
2. **Application** → **Service Workers**
3. Klicke **"Unregister"**
4. Seite neu laden

## 💡 Tipps

### Favoriten-Spiel direkt verlinken
Wenn Mama ein Lieblingsspiel hat, kannst du auch direkt darauf verlinken:
- Z.B. Lesezeichen für `https://themahjong.com/de` erstellen
- Oder Icon auf dem Desktop

### Schriftgröße anpassen
- **Größer**: `Strg` + `+`
- **Kleiner**: `Strg` + `-`
- **Zurücksetzen**: `Strg` + `0`

### Offline-Funktionalität
- Die **Startseite** funktioniert auch offline
- Zum Spielen wird Internet benötigt (externe Seiten)
