## 💡 Integration-Beispiel

### Option 1: Externe Links (aktuell)
```html
<a href="https://www.vipgames.com/de/yatzy" target="_blank">
  Yatzy spielen
</a>
```

### Option 2: iFrame-Einbettung (später)
```html
<iframe 
  src="https://embed.vipgames.com/yatzy" 
  width="100%" 
  height="700px" 
  frameborder="0"
  allowfullscreen>
</iframe>
```

**Vorteile iFrame:**
- Spiel bleibt auf unserer Seite
- Keine neuen Tabs
- Bessere UX für Senioren

**Nachteile iFrame:**
- Braucht mehr Platz
- Eventuell Registrierung erforderlich
